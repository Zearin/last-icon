Last Icon

One custom icon element to rule them all

Basic usage

Simply use <l-i></l-i>

Default icons

Set the name with name="your-icon" and, optionnaly, set a custom size with size="xx" (which is just a shortcut for setting style="--size:{xx}px")

Icons are using vertical-align:middle by default and a vertical-align:0.125em offset when contained in a p, button, a or span

Bootstrap Icons

Use set="bootstrap" or set="bs"

Box Icons

Use set="boxicons" or set="bx"

You can specify the following types: solid, regular, logos

Tabler Icons

Use set="tabler" or set="tb"

This is the default set!

You can also set various widths using stroke attribute ! It can also be set globally with defaultStroke.

Materials Icons

Use set="material" or set="mi"

You can specify the following types: baseline, outline, round, sharp, twotone

Css.gg Icons

Use set="cssgg" or set="cs"or set="gg"

Font Awesome Icons

Use set="fontawesome" or set="fa"

Use set="supertiny" or set="st"

Emoji Icons

Use set="emojicc" or set="em"

Iconoir

Use set="iconoir" or set="in"

EOS

Use set="eos" or set="eo"

You can specify the following types: solid, outlined, animated

Note: we use an alternative provider as the original repository structure is not very consistent

Feather

Use set="feather" or set="ft"

You can also set various widths using stroke attribute !

IconPark

Use set="iconpark" or set="ip"

You can also set various widths using stroke attribute !

You can set 4 themes (outline, filled, two-tone, multi-color): these need to be set in CSS.

Invalid icons

Invalid icons are logged to the console and display a warning emoji

Animations and transformations

Use the same icon and rotate

Use the animation classes or they -hover variant

Advanced usage

Host your own icons

You can point any library to a local path or to another cdn using paths.

window.LastIcon = {
  paths: {
    bootstrap: "./vendor/bootstrap",
  },
};

Change other settings

You can also configure the following settings: aliases, types, prefixes, defaultSet

window.LastIcon = {
  types: {
    boxicons: "regular",
  },
  defaultSet: "boxicons",
};
